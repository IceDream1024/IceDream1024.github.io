  2. 下载Flutter SDK

Flutter 常用技巧

命令

  1. flutter create <项目名称> - 新建Flutter项目
  2. flutter run - 运行项目
  3. flutter upgrade - 更新flutter SDK

快捷键

  1. R键： 点击后热加载，直接查看预览效果
  2. P键：在虚拟机中显示网格，工作中常用
  3. O键：切换Android和IOS的预览模式
  4. Q键：退出调试预览模式

Flutter常用组件和API

Text - 组件

用于显示一个单一样式的文本

效果图:

textAlign - 属性

textAlign用于设置文本的对齐方式, 默认TextAlign.left

  • TextAlign.center

    将文本对齐容器中心，使文字居中对齐

  • TextAlign.left

    将文本对齐容器左边，使文字居左对齐

  • TextAlign.right

    将文本对齐容器右边，使文字居右对齐

  • TextAlign.justify

    将文本拉伸至两端对齐，多余部分居文字起始方向对齐

  • TextAlign.values

    通过传入对齐方式的枚举顺序来达到与上面相同的效果，只是调用方式不同，自己喜欢哪种用哪种

    枚举顺序:
    left - 0
    right - 1
    center - 2
    justify - 3
    start - 4
    end - 5
